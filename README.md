# KVF <a href="https://github.com/seekrs/fastclass/actions/workflows/makefile.yml"><img src="https://github.com/seekrs/fastclass/actions/workflows/makefile.yml/badge.svg"></a>

KVF (kbz_8 vulkan framework) is a lightweight single-header Vulkan framework written in C99 to simplify the creation of Vulkan applications.
